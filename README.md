# Student Management System - Advanced
Folder name: sameer siddique

Features:
- PHP + MySQL web app
- Secure login (password_hash), session-based auth
- Roles: admin (can manage students)
- CRUD for students with photo upload
- Search, pagination, export CSV
- Prepared statements to prevent SQL injection
- Bootstrap 5 UI and responsive design
- Simple client-side validation

How to use:
1. Import `database.sql` into MySQL (phpMyAdmin or CLI).
2. Place project files in your webserver (e.g., htdocs or www).
3. Update `includes/config.php` MySQL credentials if needed.
4. Open `http://localhost/sameer_siddique/` and login with:
   Email: admin@example.com
   Password: admin123
